Capabilities

Everything You Need

A complete Python automation framework built for scale — from your first 2 PRs to 1,024+ and beyond.

Parallel Mode
Multi-threaded async engine processes up to 62 PRs/min with configurable concurrent workers. Built on asyncio for maximum throughput.
Token Rotation
Automatically rotates between multiple GitHub tokens, sustaining 15,000+ API requests per hour without hitting rate limits.
Proxy Support
Built-in proxy scraping from 5+ sources, concurrent testing, latency-based sorting, and automatic rotation on failure.
Live Notifications
Discord and Slack webhooks deliver real-time progress updates, success confirmations, and error alerts as they happen.
State Persistence
Interrupted runs resume automatically from exactly where they stopped. Never lose progress due to network issues or restarts.
Dry Run Mode
Simulate the full automation pipeline end-to-end without making any real commits, pushes, or pull requests to GitHub.

Quick Commands

Start in Seconds

Three commands are all you need to go from zero to Pull Shark Gold.

main.py 
# Clone & install
git clone https://github.com/itxashancode/Pull-Shark-Automation.git
cd Pull-Shark-Automation
pip install -r requirements.txt

# Authenticate GitHub CLI
gh auth login

# Sequential mode — reliable, resumable
python main.py --count 20 --delay 15
# Parallel mode — blazing fast
python parallel_automation.py \
  --count 1024 \
  --concurrent 20

# With proxy rotation enabled
python parallel_automation.py \
  --count 1024 \
  --concurrent 20 \
  --use-proxies
# Simulate without any real commits
python main.py --dry-run --count 5

# Expected output:
# 🔍 DRY RUN MODE — No commits
# 🔹 PR #1 Simulation
#   ✅ Simulation successful
# 🔹 PR #2 Simulation
#   ✅ Simulation successful

Sequential Mode

Single-threaded, reliable, and fully resumable. Perfect for small batches under 500 PRs with full control over pacing.

Parallel Mode

Async multi-worker engine hitting up to 62 PRs/min. Ideal for large batches (500–10,000 PRs) with multiple tokens.

Dry Run

Validate tokens, proxies, and webhooks without touching your repo or consuming GitHub API quota.

Installation

Get Running Fast

Four steps to set up Pull Shark Automation on any platform.

01

Clone the Repository

Run git clone https://github.com/itxashancode/Pull-Shark-Automation.git and navigate into the directory.

02

Install Dependencies

Run pip install -r requirements.txt — installs aiohttp, gitpython, requests, colorama, and more.

03

Authenticate GitHub CLI

Install gh and run gh auth login. Add personal access tokens to github_tokens.json.

04

Configure & Run

Edit config.json with your repo path, PR count, and optional webhook URLs. Run python main.py.
