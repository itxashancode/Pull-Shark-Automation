Start in Seconds

Three commands are all you need to go from zero to Pull Shark Gold.

terminal 
# Clone and install
git clone https://github.com/itxashancode/Pull-Shark-Automation.git
pip install -r requirements.txt

# Sequential mode (reliable, small batches)
python main.py --count 20 --delay 15

# Parallel mode (fast, large batches — Gold badge)
python parallel_automation.py --count 1024 --concurrent 20

# Test first with dry run
python main.py --dry-run --count 5

// installation

Get Running Fast

Four steps to set up Pull Shark Automation on any platform.

01

Clone the Repository

Run git clone https://github.com/itxashancode/Pull-Shark-Automation.git and enter the directory.

02

Install Dependencies

Run pip install -r requirements.txt — installs aiohttp, gitpython, requests, and more.

03

Authenticate GitHub CLI

Install gh CLI and run gh auth login. Add your personal access tokens to github_tokens.json.

04

Configure & Run

Edit config.json with your repo path, PR count, and webhook URLs. Then run python main.py.
